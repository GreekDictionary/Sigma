Detailed Description

This Control provides and easy way to visualise tasks. A task can be set for this TaskVisualizer, and then the progress will be automatically updated. A TaskVisualizer is visible until a ITaskObserver has been set. If the ITaskObserver is set to null, the TaskVisualizer will be invisible again.
