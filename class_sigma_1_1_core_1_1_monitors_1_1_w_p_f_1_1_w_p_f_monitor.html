void Sigma.Core.Monitors.WPF.WPFMonitor.WindowDispatcher ( Action< SigmaWindow >  action,

This method allows to access the WPFWindow. All commands will be executed in the thread of the window! (i.e. blocking until finished, use WindowDispatcherAsync<T>(Action<T>,DispatcherPriority) otherwise) If the environment has note been prepared, the function will be executed in OnStartup function of the window.

The method will block until fully executed (if the window has already been started).

◆ LogFilter

IFilter Sigma.Core.Monitors.WPF.WPFMonitor.LogFilter
getset

Every new log entry has to pass through this filter - if it passes the filter, it will get through to the root window (Window), otherwsie it will get discarded. The default is a LevelRangeFilter wth a LevelRangeFilter.LevelMin of Level.Warn.

If set to null, all messages will get passed.
