enum Sigma.Core.Training.Hooks.TargetMode
strong

The target mode for a particular hook. Used to implicitly add it to local / global hooks if specified, otherwise (if marked as TargetMode.Any require explicit call. Note: Target mode does not have to match the actual target of a hook. It's merely a recommendation.

Enumerator
Local 

The local target mode, prefer per-worker invocation.

Global 

The global target mode, prefer per-operator invocation.

Any 

The default any target mode, no preference given. Requires explicit call to attach as hook.
