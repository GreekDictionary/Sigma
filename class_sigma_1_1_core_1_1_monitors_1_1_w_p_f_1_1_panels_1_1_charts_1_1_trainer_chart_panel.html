Detailed Description

This generic SigmaPanel (should) work with every chart from LiveCharts. Depending on the generics, it plots a given chart with given data and a given amount of parameters. These parameters are automatically fetched at a given TimeStep via a ValueReporterHook.

TSeriesThe Series that is used.
