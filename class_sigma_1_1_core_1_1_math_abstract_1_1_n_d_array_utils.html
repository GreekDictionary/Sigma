Get slice indices along a certain dimension for a certain index, with all other indices set to 0 if sliceEndIndex is false or their shape limit if true.

Parameters
dimensionIndexThe dimension index, where the significant index will be set.
indexThe significant index to set.
shapeThe shape of the ndarray to be sliced.
copyResultShapeIndicate whether the resulting shape should be a copy from the passed shape or the original (modified).
sliceEndIndexIndicate whether the slice indices should be calculated as a begin index (all except significant 0) or end index (all except significant shape limit).
Exceptions
ArgumentException
Returns
The resulting slice indices, with the given index value at the significant dimension and 0 or shape limit in the others (depending on sliceEndIndex).
