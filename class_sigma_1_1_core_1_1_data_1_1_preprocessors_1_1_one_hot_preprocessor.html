Detailed Description

A one-hot preprocessor, which encodes a one-dimensional feature dimension in the one-hot format.

For example, with a given possible value range of [0, 9] 4th an input of 4 becomes [0, 0, 0, 1, 0, 0, 0, 0, 0]

7th an input of 7 becomes [0, 0, 0, 0, 0, 0, 1, 0, 0]
